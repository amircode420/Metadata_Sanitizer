# METADATA SANITIZER FOR FILE SHARING

## How to run this application

To get run this application, you must first install the required dependencies.

NOTE: This application is built on python 3.10, make sure you have python installed to run this application

run the following command in your terminal after navigating into the METADATA_SANITIZER_FOR_FILE_SHARING folder.

```bash
pip install -r requirements.txt
```
This will install all the necessary libraries.

After the dependencies are installed, 

run the following command.

```bash
python main.py
```

This will launch the application - Metadata Sanitizer For File Sharing.

This application is about removing Metadata from Files specifically (PDF, Word Document, Images) to prevent personal information leaks while sharing files online. This application provides a Clean GUI to work with and provides the user with control over their data.

Once application is launched, choose the Supported file you wish to sanitize.

The application will then display the file's metadata in the table. 

The user can choose which metadata they wish to remove from the file.

This provides the user control over their data.

After choosing what the user wants to remove, Click the "Sanitize" button. 
This will sanitize the file and save a copy of the original file, preserving the original file.

The application's operations will be logged, The logs can be viewed from the "view logs" button on the top right corner of the application.

## Users can find more information about the application by clicking the "Help" button next to the "View Logs" button.

This educates the users on what Metadata is and why it is important to remove. It is a mini guide on how the application works.
